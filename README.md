# Diet API – Python Backend Project

REST API developed in Python for managing meals and dietary information.

This project focuses on backend fundamentals such as API design, separation of concerns and clean architecture.

---

## Features

- Add meals
- List all meals
- Filter healthy meals
- Simple business rules
- RESTful endpoints

---

## Technologies

- Python 3.10+
- Flask
- REST architecture
